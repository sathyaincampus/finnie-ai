Technology Stack

Production-grade Python stack with cloud-native deployment

Layer Technology Purpose
Frontend Streamlit Multi-tab UI (Chat, Portfolio, Market, Projections, Settings)
Orchestration LangGraph StateGraph workflow, intent routing, parallel agent execution
Agents Custom Python (ABC) 8 specialized agents with BaseFinnieAgent pattern
LLM OpenAI / Anthropic / Google Adapter pattern ‚Äî hot-swap providers via UI
